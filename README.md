# LMS Video Auto Next——可用于中山大学在线教学平台

一个用于 LMS 在线课程页面的 Tampermonkey 用户脚本。

这个脚本的目标是帮助用户在视频学习页面中减少重复操作，例如自动检测视频结束、跳转到下一学习项、在跳转后尝试继续播放，以及在章节测验页面出现时提醒用户手动处理。

请遵循学校的相关规定，
对于使用该脚本产生的一系列问题作者概不负责

本作者非计算计专业学生，所有代码均由codex生成，可能存在某些未知风险。


## 功能

- 自动识别课程视频元素
- 视频播放结束后自动跳转到下一项
- 跳转到下一视频页后自动尝试继续播放
- 视频异常暂停时自动尝试恢复播放
- 支持 `dryRun` 模式，便于先测试再正式启用
- 提供右下角调试面板，可查看当前识别状态
- 支持收起/展开调试面板，避免遮挡页面按钮
- 进入章节测验页面时弹窗并发出声音提醒用户手动完成
- 记录运行日志，便于排查问题

## 不包含的功能

- 不包含自动答题
- 不包含自动选择题目选项
- 不包含自动提交测验或考试
- 不包含账号、Cookie、Token 等敏感信息管理

## 适用页面

当前脚本主要面向以下页面：

- 视频资源页：`/mod/fsresource/view.php`
- 测验说明页：`/mod/quiz/view.php`
- 测验作答页：`/mod/quiz/attempt.php`
- 测验提交页：`/mod/quiz/summary.php`

## 安装方式

### 1. 安装 Tampermonkey

先在浏览器中安装 Tampermonkey 扩展。

- Chrome / Edge：在扩展商店搜索 `Tampermonkey`
- Firefox：在附加组件商店搜索 `Tampermonkey`

### 2. 导入脚本

1. 打开 Tampermonkey 面板
2. 将 `tampermonkey-lms-autonext.user.js` 文件拖入Tampermonkey页面
3. 点击安装脚本
4. 打开中山大学在线教学平台，并刷新页面

## 使用说明

### 基本流程

1. 打开课程视频页面
2. 确认右下角出现脚本面板
3. 先保持 `dryRun: ON`，测试识别与日志是否正常
4. 确认无误后切换到 `dryRun: OFF`
5. 视频结束后，脚本会自动跳到下一项
6. 如果下一项仍是视频，脚本会尝试继续播放
7. 如果进入测验页，脚本会提醒用户手动完成

### 调试面板

面板提供以下能力：

- 重新扫描页面元素
- 切换 `dryRun`
- 手动测试“下一节”跳转
- 导出日志
- 清空日志
- 从零重置脚本状态
- 收起 / 展开面板

## 运行边界

这个脚本只用于视频页面导航和人工提醒，不具备自动化答题能力。

如果平台规则、学校要求或课程规范不允许类似脚本，请不要使用。

使用者应自行确认：

- 是否符合所在平台的使用规定
- 是否符合课程要求
- 是否符合学校或组织的相关政策
